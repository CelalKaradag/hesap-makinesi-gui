# Java Swing Hesap Makinesi

Bu proje, Java programlama dili ile geliştirilmiş basit bir grafik arayüzlü (GUI) hesap makinesidir.  
Kullanıcı, butonlar aracılığıyla dört işlem yapabilir: toplama, çıkarma, çarpma ve bölme.

## Özellikler

- Java Swing arayüzü
- Büyük butonlarla kolay kullanım
- Hatalı girişlerde uyarı (örneğin sıfıra bölme)
- C (Clear) tuşuyla sıfırlama

## Kullanım

1. Projeyi `final2.java` olarak çalıştır
2. Sayı ve işlem butonlarını kullanarak işlem yap
3. "=" tuşuyla sonucu gör

## Gereksinimler

- Java JDK 8 veya üzeri
